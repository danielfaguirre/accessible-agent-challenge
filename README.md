# Accessible Agent Challenge

## How to run

1. Open [Slingshot Studio](https://studio.slingshot.ai) and import `workflow-frontend-accessibility-challenge-workflow-*.json`.
2. Run the **`frontend-accessibility-challenge-workflow`** workflow.
3. The workflow clones this repo, reads `inaccessible.html`, and generates a WCAG 2.1 report.
4. Download the result from the **Disk** tab in the execution console — it will be saved as `accessibility-report.md`.
